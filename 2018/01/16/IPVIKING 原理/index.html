IPVIKING 分析

IPVIKING 是什么

iPviking:在线黑客攻击可视化地图有一个由挪威公司对全世界互联网数据流攻击进行的实时检测可视化数据图，用户可以直观的看到世界各地的攻击数据，这个检测平台可以监测到网络上的各种攻击方式，来源于哪个国家等详细数据。

网址：http://map.ipviking.com/ 现在已停用

新版网站：http://map.norsecorp.com/

Norse有800多万个传感器，全球的蜜罐可以在所有主要路由器附近与7PB的攻击数据进行比较和分析。它们涵盖了从一般网络流量到点对点网络到IRC网络到TOR的所有内容。收集全球范围内的Mac，Windows，ATM和网络摄像头。

实现原理

前端: HTML + CSS + JS，后端: Node.js，框架: Angular，D3js图形库

使用Websocket进行数据交互处理：ws://map.norsecorp.com/socketcluster/

地图攻击线路主要用D3js库处理。
